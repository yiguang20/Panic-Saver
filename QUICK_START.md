# 🚀 Panic Relief App - 快速开始

## 📱 立即运行应用

### 方法1：在模拟器/真机上运行

```bash
cd panic_saver
flutter run
```

### 方法2：构建 APK 安装到手机

```bash
# 构建 Debug APK（快速，用于测试）
flutter build apk --debug

# APK 位置：
# panic_saver/build/app/outputs/flutter-apk/app-debug.apk

# 构建 Release APK（优化，用于发布）
flutter build apk --release

# APK 位置：
# panic_saver/build/app/outputs/flutter-apk/app-release.apk
```

### 方法3：直接安装到连接的设备

```bash
flutter install
```

## 🎯 核心功能快速体验

### 1. 危机引导（2分钟）
1. 打开应用，看到第1步"准备"
2. 点击"准备好了"按钮
3. 依次体验9个步骤
4. 注意观察：
   - 进度点的变化
   - 第3步的呼吸球动画
   - 每步的图标和文字

### 2. 求助信（30秒）
1. 在任意步骤，**双击屏幕任意位置**
2. 观察气泡扩散动画
3. 卡片翻转显示求助信
4. 点击"返回引导"按钮回到危机模式

### 3. 练习模式（1分钟）
1. 点击底部的"PRACTICE"按钮
2. 查看4种练习类型
3. 点击关闭按钮或再次点击"PRACTICE"关闭

### 4. 多语言切换（10秒）
1. 点击右上角的语言按钮
2. 语言会循环切换：中文 → 英语 → 西班牙语 → 法语 → 德语 → 中文
3. 关闭应用再打开，语言设置会被保存

### 5. 触觉反馈（需要真实设备）
1. 点击任意按钮，感受轻微震动
2. 进入下一步，感受中等震动
3. 双击翻转，感受强烈震动
4. 切换语言，感受选择震动

### 呼吸球使用方法

**4-7-8 呼吸法：**
1. **吸气4秒** - 看着球变大，用鼻子深吸气到腹部
2. **屏息7秒** - 球保持最大，屏住呼吸
3. **呼气8秒** - 看着球变小，用嘴慢慢呼气

**为什么是4-7-8？**
- 这是经过科学验证的放松呼吸法
- 可以激活副交感神经系统
- 帮助身体从"战斗或逃跑"模式切换到"休息和消化"模式

### 求助信使用场景

**什么时候使用：**
- 在公共场合发作时
- 需要向他人说明情况时
- 无法清晰说话时

**如何使用：**
1. 双击屏幕翻转到求助信
2. 把手机递给身边的人
3. 让他们阅读求助信
4. 他们会理解你的状况并提供帮助

**求助信内容：**
- 说明你正在经历惊恐发作
- 强调你没有生命危险
- 请求不要叫救护车（除非你主动要求）
- 请求陪伴直到呼吸平稳

### 练习模式

**4种日常练习：**

1. **深呼吸** - 腹式呼吸法
   - 每天练习10-15分钟
   - 预防惊恐发作

2. **运动释放** - 消耗多余能量
   - 有氧运动
   - 减少焦虑

3. **自我对话** - 改变灾难化思维
   - 识别负面想法
   - 用理性思维替代

4. **接纳漂浮** - 威克斯漂浮法
   - 面对、接受、漂浮、等待
   - 不对抗焦虑

## 🔧 故障排除

### 常见问题

**Q: 触觉反馈不工作？**
A: 
- 检查设备是否支持触觉反馈
- 检查系统设置中触觉反馈是否开启
- 模拟器不支持触觉反馈，需要真实设备

**Q: 呼吸球动画太快/太慢？**
A: 
- 呼吸球是按照标准的4-7-8节奏设计的
- 总周期19秒（4+7+8）
- 如果感觉不适应，可以先观察几个周期

**Q: 应用闪退？**
A:
- 检查 Android 版本（需要 5.0+）
- 清除应用数据重试
- 查看日志：`flutter logs`

**Q: 语言切换后内容不变？**
A:
- 重启应用
- 检查是否所有内容都已翻译

**Q: 双击翻转不灵敏？**
A:
- 确保两次点击间隔小于300毫秒
- 不要点击按钮区域
- 点击卡片中央区域
